# 新建一篇「日常碎片」内容

## 🎯 目标

参考 `contents/daily` 目录下已有内容的结构与格式，
创建一篇以**当前日期命名**的日常碎碎念文件。

---

## 📌 要求

- 文件路径必须位于 `contents/daily`
- 文件名参考当前已存在的文件
- 内容结构参考当前已存在的文件
- 仅生成一个**最小可运行示例（Minimal Demo）**
- 不要生成额外说明文字

---

## 📝 示例内容

```md
---
title: 今日碎碎念
weather: 晴
date: 2026-02-26 15:30
---

<div>
  Hello World
</div>
```
