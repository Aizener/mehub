# 新建一篇「文章」内容

## 🎯 目标

参考 `contents/posts` 目录下已有内容的结构与格式，
创建一篇以**当前日期**和**文章主题**命名的文章文件。

---

## 📌 要求

- 文件路径必须位于 `contents/posts/{年-月}`（如 `contents/posts/2026-03`）
- 文件名格式：`post-{MM-DD}-{slug}.mdx`（如 `post-03-12-my-topic.mdx`）
- 内容结构参考当前已存在的文章
- 仅生成一个**最小可运行示例（Minimal Demo）**
- 不要生成额外说明文字

---

## 📝 示例内容

```mdx
---
title: 文章标题
summary: 文章摘要，简要描述文章内容。
date: 2026/03/12 15:30
tags:
  - tag1
  - tag2
---

正文内容从这里开始...
```
